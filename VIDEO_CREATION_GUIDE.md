## VIDEO CONCEPT OPTIONS

### Option 1: Animated Explainer Video (Recommended)
- **Duration:** 60-90 seconds
- **Style:** 2D animation with characters
- **Best For:** School students, non-technical audience
- **Tools:** Canva, Animaker, Powtoon

### Option 2: Screen Recording with Voiceover
- **Duration:** 2-3 minutes
- **Style:** Actual app demo with narration
- **Best For:** Technical audience, judges
- **Tools:** OBS Studio, Loom, Camtasia

### Option 3: Motion Graphics
- **Duration:** 45-60 seconds
- **Style:** Icons and text animations
- **Best For:** Quick attention grabber
- **Tools:** Canva, CapCut, After Effects

## FREE TOOLS TO CREATE VIDEO

### 1. CANVA (Easiest - Recommended)
**Website:** canva.com

**Why Canva:**
- Free video templates
- Drag-and-drop interface
- Built-in animations
- No learning curve

**Steps:**
1. Go to canva.com → Create → Video
2. Search "Explainer Video" templates
3. Replace text and images
4. Add animations
5. Download as MP4

---

### 2. ANIMAKER (Best for Animation)
**Website:** animaker.com

**Why Animaker:**
- Cartoon characters included
- Indian characters available
- Pre-made scenes
- Free tier available

**Steps:**
1. Sign up at animaker.com
2. Choose "Create" → "Explainer Video"
3. Select characters
4. Add voiceover
5. Export

---

### 3. CAPCUT (Best for Mobile)
**App:** CapCut (Free on Android/iOS)

**Why CapCut:**
- Completely free
- Easy to use on phone
- Good text animations
- Trending effects

**Steps:**
1. Download CapCut app
2. Import screen recordings
3. Add text overlays
4. Add transitions
5. Export

---

### 4. POWTOON (Professional Look)
**Website:** powtoon.com

**Why Powtoon:**
- Business-style animations
- Ready templates
- High-quality output
- Free educational accounts

## SCREEN RECORDING GUIDE

If you want to show actual app demo:

### Recording Tool: OBS Studio (Free)
**Download:** obsproject.com

### What to Record:

```
RECORDING SEQUENCE (2-3 minutes)

1. HOMEPAGE (10 sec)
   - Show landing page
   - Highlight "Get Started" button

2. SIGNUP (15 sec)
   - Show Google login option
   - OR show email + OTP flow

3. ONBOARDING (30 sec)
   - Enter income: 50,000
   - Select city: Mumbai
   - Family size: 4
   - Click "Generate Budget"
   - Show AI budget result

4. DASHBOARD (20 sec)
   - Pan across the dashboard
   - Show expense chart
   - Show budget vs actual

5. ADD EXPENSE (20 sec)
   - Click "Add Expense"
   - Show voice input (if possible)
   - Enter: 500, Food, Swiggy
   - Show success message

6. LANGUAGE SWITCH (15 sec)
   - Go to settings
   - Change language to Hindi
   - Show translated interface

7. DARK MODE (10 sec)
   - Toggle dark mode
   - Show theme change

8. GOALS (15 sec)
   - Show goals section
   - Add a sample goal

9. CLOSING (10 sec)
   - Back to dashboard
   - Fade out with logo
```

## TIPS FOR SCHOOL STUDENTS AUDIENCE

### DO:
- Use bright, colorful visuals
- Include relatable characters (students, young people)
- Use simple language (avoid jargon)
- Add fun animations and transitions
- Include humor if possible
- Show real Indian context (Swiggy, auto, chai)

### DON'T:
- Use technical terms (API, database, encryption)
- Make it too long (under 90 seconds)
- Use boring corporate style
- Forget background music
- Use small text (hard to see from distance)

---

## SAMPLE DIALOGUE FOR YOUNG AUDIENCE

Instead of: "Our application utilizes AI algorithms for budget optimization"

Say: "Our app is like having a smart friend who knows exactly how to save your pocket money!"

Instead of: "The system auto-categorizes expenses using NLP"

Say: "Just say 'spent 50 on chips' - the app knows it's food!"

Instead of: "AES-256 encryption protects user data"

Say: "Your secrets are safe with us - locked with the same tech banks use!"

## VOICEOVER OPTIONS

### Option 1: Record Yourself
- Use your phone's voice recorder
- Speak clearly and slowly
- Record in a quiet room

### Option 2: AI Text-to-Speech (Free)
- **Murf.ai** - Natural Indian voices
- **ElevenLabs** - Very realistic
- **Canva** - Built-in TTS

### Option 3: Get a Friend
- Someone with clear voice
- Practice the script first
- Record in one take if possible

## FINAL CHECKLIST

Before the finale, make sure:

- [ ] Video plays smoothly on loop
- [ ] Audio is audible (not too soft/loud)
- [ ] No spelling mistakes in text
- [ ] QR code is visible and works
- [ ] Video file is backed up (USB + cloud)
- [ ] Tested on actual display device
- [ ] Laptop is charged / has power backup
