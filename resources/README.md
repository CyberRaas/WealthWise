## üîÑ How to Convert Files

### 1. Markdown to PDF/DOCX
- Open the `.md` file in VS Code
- Install "Markdown PDF" extension
- Right-click ‚Üí "Markdown PDF: Export (pdf)"
- OR copy content to Google Docs/Word and save

### 2. SVG to PNG
- Open `3_Architecture_Diagram.svg` in browser
- Right-click ‚Üí Save Image As PNG
- OR use online converter: cloudconvert.com

### 3. HTML to PNG
- Open `5_Feature_Highlights.png.html` in Chrome
- Press `Ctrl + Shift + S` (screenshot)
- Select the content area
- Save as PNG

### 4. Create PowerPoint
- Open `2_WealthWise_Presentation.pptx.md`
- Copy each slide content to PowerPoint
- Add your screenshots from `public/screenshots/`
- Apply the color theme mentioned
